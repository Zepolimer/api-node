# api-node

An API with Node.js - Express - JSON Web Token (authentication)

## USING : 
```
npm install
npm start 
```

### PUBLIC ENDPOINTS :
- /users --> (GET)
- /groups --> (GET)
- /groups/users --> (GET)
- /register --> (POST)
- /login --> (POST)

### PUBLIC ENDPOINTS :
- /users/:id --> (GET, PUT, DELETE)
- /users/:id/groupe --> (PUT)
- /groups --> (POST)
- /groups/:id --> (PUT, DELETE)
